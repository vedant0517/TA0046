# CareConnect

A React-based healthcare management platform.

## Getting Started

### Installation

```bash
npm install
```

### Running the Application

```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)

### Building for Production

```bash
npm run build
```

## Features

- Responsive navbar with CareConnect branding
- Home page with hero section
- Service features showcase
- Modern UI with custom CSS (no Tailwind)
